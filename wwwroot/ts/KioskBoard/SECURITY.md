# Security Policy

## Supported Versions

KioskBoard is a dependency-free JavaScript library that has no vulnerabilities. Supported versions can be found in the table below.

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0.0 | :x:                |
